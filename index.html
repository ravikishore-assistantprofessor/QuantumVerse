Overview

About the Course

Designed for undergraduate students in science and engineering. This course bridges the gap between quantum mechanics principles and practical software implementation on quantum simulators.

Course Objectives

Develop a foundational working knowledge of quantum computation. By completing this program, students will transition from abstract mathematical spaces to designing fully functional algorithms in Qiskit.

Prerequisites

  • Linear Algebra (complex vector matrices, eigenvalues)
  • Basic Probability Theory
  • Introductory Python programming

Skills Gained

  • ✓ Single & multi-qubit transformations
  • ✓ Entanglement manipulation
  • ✓ Qiskit circuit modeling
  • ✓ Quantum cryptography basics

1. Qiskit Installation & Setup Guide

It is recommended to run Qiskit locally rather than in the browser to maintain access to local plotting backends like Matplotlib.

Step 1: Install Python Virtual Environment

python -m venv qiskit-env
# Activate on Windows Powershell:
.\qiskit-env\Scripts\Activate.ps1

Step 2: Install Qiskit via pip

pip install qiskit qiskit-aer matplotlib pylatexenc

2. Your First Quantum Program: Hello Quantum

Copy and paste this script into a local Python file (e.g. hello_quantum.py) and run it using your terminal.

# hello_quantum.py
from qiskit import QuantumCircuit
from qiskit_aer import AerSimulator

# Create a 2-qubit register circuit
qc = QuantumCircuit(2)

# Apply H to qubit 0 (Superposition)
qc.h(0)
# Entangle qubits
qc.cx(0, 1)

# Add measurements
qc.measure_all()

# Simulate
simulator = AerSimulator()
result = simulator.run(qc).result()
counts = result.get_counts()

print("Measurement outcomes:", counts)

FAQ

Frequently Asked Questions

Find quick answers to academic parameters, assignments, exams, and laboratory setups.

Who is this course designed for?

Undergraduate engineering students majoring in Computer Science, Electrical Engineering, or Physics. Minimal prior exposure to quantum mechanics is assumed; all linear algebra is reviewed in Module 1.

Which version of Qiskit does the curriculum support?

The course syllabus conforms strictly to Qiskit v1.0+ syntax specifications. Old deprecated imports (like Execute and qiskit.providers) are replaced with modern AerSimulator execution wrappers.

Is this course accredited under standard university frameworks?

Yes, this portal maps directly to standard 4-credit course syllabi for semester-long Fundamentals of Quantum Computing courses in engineering colleges.
